## 🚦 Hard rules (footprint avoidance)

These are the actual signals Google uses to detect "owner networks." Break any one and the whole campaign turns into a penalty vector:

1. **Never sitewide / footer / sidebar links** — only in-article editorial mentions inside genuine content.
2. **Never same anchor text twice in a row** — rotate brand / URL / generic / keyword. Aim for max 25% any single anchor across all 12 months.
3. **Never same target URL twice in a row** — rotate across pillars + audience pages.
4. **Never 2+ projects link the same day** — minimum 7-day gap between any two sister-project links to MerrySails.
5. **Always topical relevance** — link only fits if the article is about Istanbul travel / tourism / cruise / yacht / event planning. A car rental site can link a cruise via "Istanbul trip planning" angle; a tattoo site cannot.
6. **Always alongside neutral mentions** — if the article mentions MerrySails, it should also mention at least one neutral comparison (Şehir Hatları ferry, generic "TÜRSAB-licensed operators", etc.). Single-brand promotion looks paid.
7. **Always real visual integration** — embed a real MerrySails photo (operator-owned, EXIF intact) into the linking article. Stock + link = obvious link bait.
8. **Always dofollow first** — Google weights dofollow editorial links higher than nofollow. Use nofollow only if the article is labeled `sponsored:` or `affiliate:`.
9. **Never auto-generate** — every article hand-written by the project's existing editorial voice. AI-generated bulk = scaled content abuse signal.
10. **Never spam GSC URL inspection** — submit only the linking article, not the MerrySails target URL. Let the natural crawl pick up the new link.

## ⚠️ Stop conditions

Cancel the next planned link if any of these happen:

- **Babbar score >40** (footprint detected)
- **GSC manual action** notice (link scheme penalty)
- **MerrySails impressions drop >30% week-over-week** unexplained
- **Two projects in same week** accidentally link the same target URL (means a 90-day cool-down for both projects)
