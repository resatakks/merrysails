# MerrySails Ad Schedule And Geo Plan

Use this plan for the first launch. Do not broaden until search-term quality is clean.

## Campaign Schedule

Account time zone should be Istanbul.

Launch schedule:

```text
(Monday[08:00-23:30]);(Tuesday[08:00-23:30]);(Wednesday[08:00-23:30]);(Thursday[08:00-23:30]);(Friday[08:00-23:30]);(Saturday[08:00-23:30]);(Sunday[08:00-23:30])
```

Reason:

- Tourists search before hotel breakfast, during daily planning, and after dinner.
- WhatsApp / call response quality is strongest when the team is awake.
- Overnight curiosity clicks are less useful for the 1,000 TL/day test budget.

If same-day Dinner demand becomes strong, keep Dinner live until `23:30`; do not cut evening search too early.

## Location Setting

Core setting:

- Target: Istanbul
- Location option: Presence only
- Exclude: none at launch unless reports show waste
- Language: English
- Network: Google Search only
- Search Partners: off
- Display expansion: off

## Priority Demand Zones

Watch location and user behavior around:

- Sultanahmet / Blue Mosque / Hagia Sophia hotel cluster
- Sirkeci / Eminonu / Gulhane
- Karakoy / Galata / Tophane
- Taksim / Istiklal / Cihangir
- Beyoglu hotel corridor
- Besiktas / Dolmabahce / Kabatas
- Ortakoy / Ciragan / Kurucesme
- Nisantasi / Harbiye / Sisli
- Bebek / Arnavutkoy for premium yacht intent
- Levent / Maslak for corporate yacht intent

## Hotel Logic

High-value visitor clusters:

- 4-star and 5-star Bosphorus / old-city / Taksim hotels.
- Strong 3-star Sultanahmet and Sirkeci hotels where guests buy activities from reception.
- Boutique hotels in Karakoy, Galata, Cihangir, Sultanahmet, and Sirkeci.

Google Search campaign location targeting should not start with hotel-name or exact hotel-radius fragmentation because the budget is small. Use location reports first, then add radius layers only when data proves a zone wins.
